# Die Abenteuer der Gegelle Boys

Mobile-first RPG Partyspiel fuer die Gegelle Boys.

## Starten

Die App ist eine statische Web App. Lokal reicht ein kleiner Webserver im Projektordner, zum Beispiel:

```powershell
python -m http.server 5199
```

Dann im Browser oeffnen:

```text
http://127.0.0.1:5199/
```

## Inhalt

- Story Packs mit Kapiteln, Quests und Minispielen
- Spieler-Setup fuer bis zu 12 Personen
- Klassenwahl mit Avataren und Skills
- TV-/Tablet-taugliche Spielansicht
- Offline Cache per Service Worker
